void Battle::Fight ( bool  player_attacks)

Calculates a battle round.

Updates contenders stats making them attack each other and updates the log. If the player has won, it updates their xp & gold

Parameters
player_attacksIf false, only the enemy attacks the player, and not the other way around.

void Battle::Flee ( )

Makes the player try to run away from battle.

It has a 30% chance of the enemy attacking the player and updating the log. It has a 70% chance of ending the battle and returning to the Game.

void Character::Update ( int  tile_width,
int  tile_height 
)

Updates character animation.

Sets the current sprite to the next on the current_animation_ and advances said sprite in the map proportionally to the map's tile size. E.g. If a tile is 8px side and there's 8 frames in the current animation, it'll advance 1px per frame.
